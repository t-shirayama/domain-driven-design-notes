# ユビキタス言語

ユビキタス言語は、業務担当者と開発者が同じ意味で使う共通言語です。会議、仕様書、コード、テストで同じ言葉を使うことで、言葉のズレを早く見つけます。

単語を英訳してクラス名にするだけでは不十分です。その言葉がどの業務文脈で、どの状態やルールを含むのかまで合わせる必要があります。

| 悪い状態 | よい状態 |
| --- | --- |
| 同じ `Order` が部署ごとに違う意味を持つ | 文脈ごとに言葉の意味を確認する |
| コードでは `Data` や `Info` が多い | 業務語で名前を付ける |
| 仕様とコードの語彙が離れている | テスト名にも業務語を使う |

**ユビキタス言語は用語集ではなく、モデルを育てるための会話の道具**です。
